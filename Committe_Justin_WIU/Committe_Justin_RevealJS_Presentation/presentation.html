Tooltipster

"A powerful, flexible jQuery plugin enabling you to easily create semantic, modern tooltips enhanced with the power of CSS."
Developed and maintained by Caleb Jacob and Louis Ameline under the MIT license

Presentation by Justin Committe

Tooltipster allows you to do things many things, such as:

  • The default settings
  • Custom hide/show triggers
  • Custom themes and animations
  • Interaction with tooltips
  • Multiple tooltips on a single element
  • Fixed width, position and HTML
  • Disable touch devices
  • Custom callbacks
  • Attach tooltips to icons

Setting it up in the <head> of your document

First you need to load the stylesheets for Tooltipster and any themes you will use.

<link rel="stylesheet" type="text/css" 
href="../css/themes/tooltipster-noir.css"/>
<link rel="stylesheet" type="text/css" 
href="../css/themes/my-custom-theme.css"/>
<link rel="stylesheet" type="text/css" 
href="../css/tooltipster.css" />
Next load jquery and then Tooltipster Script files.

<script type="text/javascript" 
src="http://code.jquery.com/jquery-1.7.0.min.js"></script>
<script type="text/javascript" 
src="../js/jquery.tooltipster.js"></script>
The final thing we need to do is activate Tooltipster and add our own javaScript script link.
You can use any selector you would like to use, in this case we're using .tooltip

<script type="text/javascript">
		$(document).ready(function() {
			$('.tooltip').tooltipster();
		});	
</script>

<script type="text/javascript" 
src="../js/script.js"></script>

Examples

Custom Theme

HTML

<section id="myMap" class="container" title="My Custom Theme">
        
</section>
JavaScript

$('#myMap').tooltipster({
        content:'my custom theme',
        theme: 'my-custom-theme'
    });

CSS for custom theme

CSS needs to be in the themes folder in a custom theme file.

.my-custom-theme {
    border-radius: 10px;
    border: 3px solid blue;
    background: lightBlue;
    color: white;
}
.my-custom-theme .tooltipster-content {
    font-family: Arial, sans-serif;
    font-size: 18px;
    line-height: 20px;
    padding: 12px 15px;
}

Simple tool tip


<h1 class="tooltip" title="my H1">My ToolTipster Example</h1>

Simple tooltip added to a link


<a href="http://iamceege.github.io/tooltipster/" class="tooltip" 
title="Tooltipster Link">Tooltipster</a>

Tooltip with a different theme and HTML added through javaScript


<div class="container" id="my-tooltip">
        You can even embed html inside of your tooltips, which can give you images!!!
</div>

Javascript


$('#my-tooltip').tooltipster({
    content: $('<span><img src="../images/baby.jpg" /> 
    <strong>This is my baby, Victoria, in a different theme called noir
    </strong></span>'),
    theme:'tooltipster-noir'
});
